# Smart Resume Analyzer

Smart Resume Analyzer is a web-based application that analyzes resumes and matches them with job descriptions using Natural Language Processing (NLP) and Machine Learning techniques. It provides a resume score and section-wise feedback to help candidates understand their job fit and improve their resumes.
